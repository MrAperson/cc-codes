Attributes

Photo
  1. br tag (self-closing tag)
    • br tag is for line breaking.
    • It is used to break one line to another
    • <br />
  2. form tag
    • form tag is for creating sumbitable forms
    • Every form elements will be written in form tag.
    • <form>Elements</form>
  3. lebel tag
    • label tag is used for labeling our form questions.
    • label's name will go inside opening tag and closing tag.
    • <label>Name</label>
  4. input tag (self-closing tag)
    • input is used for asking users what their answer is.
    • It is basically like questioning.
    • input tag has some attributes, more on that later
    • <input />
  5. select tag
    • select tag is a tag that the program gives the user an optional bar.
    • select tag comes with another tag inside it, more on below..
    • <select>Another Tag</select>
  6. option tag
    • option tag is a tag that we use with select tag.
    • It provides options for users.
    • We can use that like this,
      <select>
      <option>
      Some Content inside
      </option>
      </select>
  7. textarea tag (self-closing tag)
    • This tag is a tag that gives a message box to the user that is writable.
    • The area of the box is adjustable.
    • All we need to do is to type
      <textarea />
      and the text editor will take care of the rest.
  8. (thead,tbody,tfoot) tag
    • Those three tags are all table tag's elements.
    • Apparently, thead means table head and every head part of the table will go into it.
    • tbody(table body) is for body part of table.
    • tfoot stands for table foot and it takes the bottom part of the table
    • thead - <thead>Head Part</thead>
    • tbody - <tbody>Body Part</tbody>
    • tfoot - <tfoot>Bottom Part</tfoot>
  9. th tag
    • th tag
      <th> </th>
      is a tag used inside thead tag.
    • th tag is the element of thead tag.
  10. button tag
    • As its name, button tag gives us uttons.
    • In html form, its:
      <button>Button Name</button>
  1. type=""
    • type attribute provides types of inputs.
    • So far, we've learned "email", "password", "text", "radio", "checkbox"
  2. placeholder=""
    • placeholder attribute will show placeholder texts on inputs.
    • For example, if we want to tell the user a hint for filling email input, we can use place holder like this:
      <input type="email" placeholder="xxx@xxx.com">
  3. autocomplete=""
    • autocomplete attribute thakes exactly two arguements, on and off.
    • The program will decide either give the user suggestions or don't.
  4. name=""
    • name is something like authority.
    • If we give the same name to two different checkbox or radio, whatever type of optional inputs, the program will only allow one option for the user to select. Different name, multiple slection is available.
  5. required
    • This attribute will make our inputs inevitable.
  6. id="", for=""
    • id and for can be used to link our inputs with their respective labels.
    • It allows users to reach the input by clicking either label or input.
    • Example,
      <label id="male">Male</label>
      <input type="radio" for="male" />
  7. value=""
    • value take care of some backend processes
