Selector types

So far, we have learned 3 types of selector:

  • Element Selector
  • id selector
  • class selector
  1. element selector
    • tagName {
      css property and value pairs
      }
  2. class selector
    • .className {
      css property and value pairs
      }
  3. id selector
    • #idName {
      css property and value pairs
      }

    4. We called those curly braces scopes.



External Style

    <link href="Dir/file.extension" rel="stylesheet" >

  2. To get out of our locat dir, we will use double-dot and backslashes to escapse from

Block Elements or Inline Elements

In world of html, there are two types of html tags:

  1. Block Elements
    • Elements that takes full position of where they located are called block
      elements.
    • Block elements list
      • div
        <div> Content </div>
      • table
        <table> Content </table>
      • td
        <td> Content </td>
      • h1 - h5
        <h1> Content </h1>
      • p
        <p> Content </p>
      • li
        <li> Content </li>
  2. Inline Elements
    • Elements that takes only their element position of where they located are
      called inline elements.
    • Inline elements list
      • i
        <i> Content </i>
      • u
        <u> Content </u>
      • button
        <button> Content </button>
      • ul
        <ul> Content </ul>
      • ol
        <ol> Content </ol>

Box Model

There are three parts in the term "box model"

  1. Margin
    • Margin is something that the elements take the space between them and ano-
      ther elements.
    • Margin has four parts: top, bottom, right, left
    • Depends on our margins value, we will get our elements distance.
  2. Padding
    • Padding is the space between our content and border
    • Padding also has four parts: top, bottom, right and left
    • Based on our padddings value, we will get our content's aura
  3. Border
    • I should have introduced border before I intorduce paddings.
    • Border is just like a fence for our content.
    • It provide us a border for our content.
