Tags

1. a tag (anchor tag)

- a tag <a> </a> is a tag that we can add some link into our webpage. a tag has an attribute in the opening tag, href="". We add a text that takes us to our link addresss. Thus, we use like this: <a href="Address"> TextThatRefersOurLink </a>

2. video tag

- Like its name, video tag helps us to display videos on our webpage. Video tag has its own attributes such as controls, loop and so on. It comes with src attribute too. Here's how it can be finished, <video src="Address" controls width="" poster="" > </video>

3. audio tag

- Similarly, audio tag has same attribute as video tag. I will Just show how we can use that. <audio src="Address" controls,etc.. > </audio>

4. (table,tr,td) tag

- I will show everything is just one paragraph. As their name, we can use them to build tables. we use table tag to build tables, tr tag is for table rows, td means table data. we add attributes in ttable tag. So we write lke this. <table **attributes** > <tr> <td> </td> </tr> </ttable>

5. list tags (ol,ul,li)

- ol tag <ol> </ol> means ordered list, it will show a list in numbered order, e.g. 1, 2, 3. ul tag <ul> </ul> means unordered list, it will list up its items with bulletpoints. To add items, we use <li> </li> tag and our content in between.

6. Linking YouTube videos and Map

- I will demonstarte in step by step instructions. Click Any YT video -> share -> embed -> copy embed code -> paste in your code. Those links are customizable. For example, width, height, more on that later.

Attributes

  • border - can add border lines in table width - can adjust width in any possible tag
  • height - can adjust height in any possible tag
  • poster - works with videos tag, like a thubnail
  • loop - can loop videos or audio
  • controls - can add to videos and audios to see controls
  • cellspacing - cellspacing is used in table tag for space between two data
  • cellpadding - cellpadding is used in table for space between data and border
  • target - can use in a tag. In target attribute, _blank is to jump to anoter tab, _self will run inside our webpage
