2. Tags that we use in a html file

x
1. h tag

- First, we will start with h tag. It is used in html with opening and closing tags. (<h1>) is opening tag, (</h1>) is closing tag, we write our heading name between opening tag and closing tag per <h1> Heading </h1>.We use h tag to write headings of paragraphs like titles. We use h tag in 6 ways. h1 is the biggest one and get smaller than smaller per h2, h3, h4, h5, h6. We can see that h1 is biggest while h6 is the smallest. ** note that the word (title) here is not a syntax, more on that later.

2. p tag

- Second, we will use p tag in order to write some paragraphs on our webpage. We use p tag by <p> Our Paragraph </p>. We can clearly see that p tag also has both opening and closing tags.

3. lorem

- Third one is something called lorem. lorem is a free website that gives us some random paragraphs to test our html programs. In this case, the p tag. We can easily get paragraphs from lorem by simpliy googling the word "lorem" and get along with the website. Luckily, we can get lorem's paragraphs in html file automatically without any necessary of browsing the lorem website. **We will see that when we use VScode text editor, more on that later.

4. head tag and body tag

- I am going to write both head tag in body tag in the same paragraph because they have the simple basic principles so far. The head tag is a tag we used to describe our program's tab part when we run the code in the browser. <head> Codes Of The Head Part </head>. The body tag is used when we want to show our full webpage on the website, this is why its called the body part. We can use body tag per <body> Codes Of The Body Part </body>.

5. html tag

- Like its name, the html tag is used whenever we want to write html codes, which means everything will be written between html tag. e.g, <html> <head> <\head> <body> </body> </html>

6. title tag

- We can name our tab with title tag and some content in between with <title> Tab Name </title>. The title tag will be written in <head><\head>(tab part) of our code.

7. hr tag

- The hr tag is a tag that doesn't have a closing tag, which we called self-closing tag, meaning we don't need to write any content in between. The usage of hr tag is texting <hr\> to draw a horizontal rule or line to separate our contents with a black line.

8. img tag

- The img tag is a tag we can use to add some images to our codes. <img src="">. We can see that img tag is one of the self-closing tags as it doesn't have any closing tag. But here we can see something strange inside the img tag, src="", that's the attribute of the img tag where we can add our images between two double quotes. If we are adding a local photo, we just simply use the attribute like this, <img src="./imagefilename.format">. Or if we are adding a photo from a brower or whatever online, we just need to add the image address inside the src attribute per <img src="imageAddress">. **note that we only need to add (./) if we want to use a local photo and not necessary in internet based photo.**also note that local photo should be in the same folder of our html file.

9. link tag

- Lastly, we will use the link tag in our head part so that we can see what appear on our tab part of our webpage. link tag aslo is a self-closing tag, <link rel="" href="">. The href attribute is used to add some local image or online image address. The rel attribute is used to identify what type of usage we want to do that with that photo, we will type 'icon' so that we will be able to see that photo on our tab icon. The link tag is written between head tag. e.g, <head> <link rel="icon" href="FileAddress"> </head>.

10. !DOCTYPE tag

- This is where we declare our file type. It is written at the top of our code. We are coding in html, so we will type like this, <!DOCTYPE html>
